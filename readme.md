# SmartCalendar
Представляет собой графический интерфейс календаря, который позволяет создавать события с определённой датой.  
## Запуск:
1. (cmd/bash): `cd (ProjectPath)/Qt_SmartCalendar`
2. `pip install -r requirements.txt`
3. `main.py`
